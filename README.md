# MultiObjective-HFS


## 各目录和文件说明

- `data/raw/`  
  存放原始数据文件，未经任何处理，保持数据原貌。

- `data/processed/`  
  存放预处理后生成的数据文件，便于快速加载和后续分析。

- `src/feature_selection.py`  
  实现了 MOO-HFS 多目标混合特征选择算法的核心代码。

- `src/utils.py`  
  包含数据加载、结果评估、绘图等辅助函数。

- `src/main.py`  
  项目主程序，负责调度算法运行、参数设置及结果输出。

- `experiments/`  
  各类实验脚本，包含不同参数组合和实验流程。

- `notebooks/`  
  Jupyter Notebook 文件，展示交互式分析和可视化结果。

- `requirements.txt`  
  项目依赖的第三方库，方便环境复现。

- `README.md`  
  项目整体介绍、运行方法及注意事项。

- `LICENSE`  
  项目开源许可协议，说明授权使用范围。

- `setup.py`  
  （可选）Python 安装脚本，用于打包和发布。
